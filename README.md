# landcover-sentinel2

> Sentinel-2 land-cover classification as a proper study: the original Random-Forest classifier kept as a baseline and compared against a U-Net and a LoRA-fine-tuned geospatial foundation model on identical, spatially blocked splits — plus a DINOv3 linear probe on EuroSAT and label-efficiency curves.

🚧 **Modernization in progress (July 2026).** Originally built Fall 2024 as an independent project; being published and extended here with tests, CI, and reproducible results. Code landing soon.
